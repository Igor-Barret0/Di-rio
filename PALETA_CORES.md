# 🎨 Paleta de Cores - Diário Emocional

## 📋 Briefing do Projeto
- **Público-alvo**: Jovens (14-25 anos)
- **Objetivo**: Acolhimento + Dinamismo
- **Contexto**: Saúde mental, ansiedade, segurança emocional
- **Vibe**: Moderno, profissional, mas não robótico

---

## 🎪 Guia de Uso das Cores por Contexto

### **Acolhimento** (Quando o usuário está em situação de risco/tristeza)
- Use **Roxo (#A855F7)** para confortar
- Combine com **Verde (#10B981)** para esperança
- Evite vermelho, use tons suaves

### **Dinamismo** (Chamadas à ação, celebrações)
- Use **Azul (#4F46E5)** para CTAs principais
- Use **Rosa Coral (#F97316)** para destaques secundários
- Combine com animações suaves

### **Confiança** (Admin, dados, segurança)
- Use **Azul (#4F46E5)** como principal
- Use tons neutros para tabelas
- Use **Vermelho** apenas para ações destrutivas

### **Clareza Emocional** (Emoções do usuário)
- Cada emoção tem sua cor exclusiva
- Deixa visual e intuitivo
- Usuário sabe o que sente apenas vendo a cor
